
# Inventory Management 

A python tool for basic item magment.
made small teams, that need basic and simple app for managint their equipmenet

##Features
*Add new item to your system
*Delete items using their ID.
*Get ststus after every change

##Tech Stack
* **Language** Python 3.14.3
* **DataBase** none
* **Libraries** tabulate, colorma

## Installation and Setup

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/inventory-system.git
```

2. **Navigate to the project directory:**
```bash
cd inventory-system
```

3. **Run the application:**
```bash
python main_file.py
```

## Usage

Here is a quick example of how to interact with the system once it's running:

```text
> Select action (add/search/view): add
> Enter item name: APC Smart-UPS SRT 5000VA
> Enter quantity: 2
> [Success] Item added to inventory!

> Select action: search
> Enter keyword: Eaton
> Found: 1x Eaton UPS in stock.
```
